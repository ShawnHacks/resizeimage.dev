## 三、使用浏览器自带功能截图

### 1. Chrome 浏览器

Chrome 内置了 **开发者工具（DevTools）**，可以直接生成全页截图。

操作步骤如下：

1. 打开你想截图的网页。
2. 按下 `F12` 或 `Ctrl+Shift+I` 打开开发者工具。
3. 在右上角点击“三个点” → 选择 **运行命令（Run Command）**。
4. 输入 `screenshot`，选择 **Capture full size screenshot**。
5. Chrome 会自动保存完整网页截图。

这种方法无需安装额外插件，但步骤稍显复杂。

### 2. Microsoft Edge

Edge 与 Chrome 相似，也支持全页截图：

1. 打开网页。
2. 按下 `Ctrl+Shift+S`，选择 **全页截图**。
3. 保存即可。

不过部分网页可能截图不完整。

## 四、浏览器插件：最便捷的选择

如果你经常需要截图网页，推荐使用插件。

常见插件包括：

* **GoFullPage**（Chrome 插件）

安装后，只需点击插件图标，就能一键保存完整网页，支持 JPG、PNG、PDF 格式。

但缺点是：

* 需要安装额外扩展，占用浏览器资源。
* 部分功能需要付费解锁。
